# Scram-rs

A SCRAM-SHA1 and SCRAM-SHA256 client and server.  

Supports:  
- SHA-1 hasher
- SHA-256 hasher
- SHA-512 hasher
- Client
- Server Channel Binding TLS-Server-Endpoint 256, 512

Does not support:
- Client channel binding
- authzid (a=)

Based on:  
- pbkdf2
- sha2 sha-1
